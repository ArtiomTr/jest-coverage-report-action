Upgrading to v2

Jest Coverage Report action v2 resolves the architectural issues of the first version, which prevented the addition of new functionality. In addition, second version's behavior is more predictable and understandable.

Actually, you need to update only your workflow's configuration.

Upgrade steps

A few steps need to be completed to fully migrate from v1 to v2. Don't worry if your action unexpectedly fails - this guide will help you to fix all errors. If after completing this guide your action still failing, please create an issue - our community is ready to help you!
